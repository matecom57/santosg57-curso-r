Ejercicio
=========

A partir del archivo de datos 'EXA_C01_S04_01.csv' del libro-Daniel que tiene las
edades de sujetos, consideremos "X" como la variable aleatoria de las edades. Calcular lo siguiente, por dos
métodos diferentes, (1) utilizando la funcion de distribución normal y (2) la frecuencia relativa.

1. :math:`P(X <= 40)`

2. :math:`P(X <= 60)`

3. :math:`P(X > 65)`

4. :math:`P(40 <= X <= 60)`
