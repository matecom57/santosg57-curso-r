Elementos del Lenguaje
======================

- Es un lenguaje interpretado; las expresiones especificadas en un programa R se ejecutan línea por línea de manera similar a otros lenguajes interpretados, 

- Tiene tipos dinámicos, lo que significa que infiere los tipos de datos de las variables en función del contexto. No necesitamos declarar las variables por separado.

- Maneja tipos básicos de datos: números enteros, números de punto flotante, números complejos y cadesnas de carácteres (string's).

- Tiene constantes.

- Tiene operadores aritméticos, de comparacion, lógicos.

- Tiene estructuras de control.

- Tiene estructuras de datos, que identifica este tipo de dato como una unidad de información.
